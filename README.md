# 全领域开放数据集

> 建立一个覆盖多学科的高质量数据集索引平台，帮助科研人员与工程师快速定位合适数据源。

本项目旨在收集、整理与索引各领域的公开数据集，
覆盖计算机视觉、自然语言处理、语音、多模态、遥感、医疗等方向。
